# MHAP

MinHash alignment process (MHAP pronounced MAP): locality sensitive hashing to detect overlaps and utilities. This is the development branch, please use the [latest tagged](https://github.com/marbl/MHAP/releases/tag/1.6).

## Build

You must have a recent  [JDK](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html "JDK") and [Apache ANT](http://ant.apache.org/ "ANT") available. To checkout and build run:

    git clone https://github.com/marbl/MHAP.git
    cd MHAP
    ant
    
For a quick user-quide, run:

    cd target
    java -jar mhap-1.6.jar
